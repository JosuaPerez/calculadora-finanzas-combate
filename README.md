# 🛡️ Calculadora Financiera - Finanzas de Combate

Una herramienta web ligera, directa al grano y sin dependencias, diseñada para organizar tus finanzas quincenales o mensuales. Creada para quienes necesitan un plan de acción claro para liquidar deudas rápidamente ("Modo Guerra") o construir ahorros sólidos ("Modo Construcción").

🌍 **Prueba la aplicación en vivo aquí:** [https://calculadora-financieraa.netlify.app/](https://calculadora-financieraa.netlify.app/)

## ✨ Características Principales

* **Cálculos en Tiempo Real (RD$):** Olvídate de calcular porcentajes a mano. Ingresa los montos exactos de tus gastos y la calculadora te mostrará instantáneamente qué porcentaje de tus ingresos representan.
* **Validación Anti-Quiebra (Sobrante Cero):** Si tus gastos fijos igualan o superan tus ingresos, el sistema bloquea visualmente la distribución de sobrantes y te lanza una alerta. ¡Nada de inventar dinero que no existe!
* **Categorías 100% Personalizables:** Añade, edita o elimina gastos fijos y categorías de distribución. Ajusta los conceptos a tu realidad (diezmo, gasolina, fondo de emergencia, etc.).
* **Exportación a Excel (CSV):** Genera y descarga un reporte estructurado en formato `.csv` con un solo clic. Ideal para pasar tu presupuesto directamente a tu matriz de control de deudas en Excel o Google Sheets.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla Puro).
* **Cero Dependencias:** El proyecto fue optimizado para no depender de ninguna librería externa, haciéndolo ultra rápido y ligero.

## 🚀 Cómo usar este proyecto en local

1. Clona este repositorio en tu máquina local:
   `git clone https://github.com/JosuaPerez/calculadora-finanzas-combate.git`
2. Ve a la carpeta del proyecto.
3. Abre el archivo `index.html` en cualquier navegador web.
4. ¡Empieza a presupuestarte!
